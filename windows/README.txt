

Run "INSTALL_WEB_SERVER_FIRST" to setup the web server BEFORE RUNNING Open Crypto Tracker.


Run "RUN_CRYPTO_TRACKER" to run Open Crypto Tracker.


Run "UNINSTALL_WEB_SERVER" before deleting this app folder (if you want to remove this app).


Run "ADD-WIN10-SCHEDULER-JOB" to run background tasks with Windows Task Scheduler (instead of in the web browser). You'll need to disable browser-based background tasks in the admin config. See the Help / FAQ page section labeled "Setting Up Price Charts And Email / Text / Telegram / Alexa Price Alerts" for more information.
